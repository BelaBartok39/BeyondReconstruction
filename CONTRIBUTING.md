# Contributing to BeyondReconstruction

Thank you for your interest in contributing!

## How to Contribute

1. **Fork** the repository on GitHub.
2. **Clone** your fork locally:
   ```bash
   git clone https://github.com/<your-username>/BeyondReconstruction.git
   ```
3. **Create a branch** for your changes:
   ```bash
   git checkout -b feature/your-feature-name
   ```
4. **Make your changes** and add tests where appropriate.
5. **Run the tests** to make sure nothing is broken:
   ```bash
   pytest tests/ -v
   ```
6. **Commit and push** your changes, then open a Pull Request.

## Code Style

- Follow [PEP 8](https://peps.python.org/pep-0008/) conventions.
- Add type hints consistent with the existing codebase.
- Use double quotes for strings.

## Reporting Issues

Please open an issue on [GitHub](https://github.com/BelaBartok39/BeyondReconstruction/issues) with a clear description of the problem and steps to reproduce it.
